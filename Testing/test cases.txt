1. Проверить фильтрацию по прямым рейсам
	- зайти на сайт https://santufei.com
	- ввести нужный маршрут (например, Минск - Москва)
	- нажать на кнопку поиска
	- поставить галочку на пункте "без пересадок"
   Ожидаемый результат:список рейсов без пересадок.
   Результат: найдено 15 рейсов Из Минска в Москву без пересадок.

6. Проверить, что нельзя выбрать только ребенка без врослого
	- зайти на сайт https://santufei.com
	- ввести нужный маршрут
	- попробовать убрать из списка пассажиров всех взрослых и добавить ребенка
   Ожидаемый результат: невозможность заказать билеты для детей без взрослых.
   Результат: поле с количеством взрослых не может быть меньше 1.

7. Проверить, будут ли предлагаться все аэропорты при вводе какого-либо города
	- зайти на сайт https://santufei.com
	- ввести город отправления
	- ввести город прибытия
   Ожидаемый результат: при вводе городов в списке появляется несколько (если они есть) аэропортов.
   Результат: для Москвы появляются все аэропорты, а так же есть возможность выбрать просто город, после чего будут отображены рейсы для всех аэропортов.

8. Проверить заказ билета в одну сторону
	- зайти на сайт https://santufei.com
	- ввести город отправления и прибытия
	- выбрать пункт в одну сторону
   Ожидаемый результат: невозможность выбора обратной даты.
   Результат: поле с датой обратного отправления становится неактивным.

9. Проверить заказ билета в обе сторону
	- зайти на сайт https://santufei.com
	- выбрать пункт в обе стороны
	- ввести города отправления и прибытия
	- ввести даты "туда" и "обратно"
   Ожидаемый результат: невозможность выбора обратной даты ранее даты отправления.
   Результат: даты ранее даты отправления для даты "обратно" недоступны.

10. Проверить работу функции "История поиска"
	- зайти на сайт https://santufei.com
	- провести поиск нескольких маршрутов
	- вернуться на главную
	- нажать на кнопку "История поиска"
   Ожидаемый результат: список с маршрутами, которые искали ранее.
   Результат: отображается список с маршрутами, которые искали ранее, с их датами, количеством пассажиров и классом.
